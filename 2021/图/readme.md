# 图

## 图是什么
Graph 图是一种非线性数据结构，由 vertices（顶点）和 edges（边）组成。

## 图的应用场景
图的应用场景非常广泛，比如社交网络、地图、网络拓扑等。

## 图表示
    B
  /   \
A       C

Adjacency Matrix 邻接矩阵
```js
[   A  B  C 
  A[0, 1, 0],
  B[1, 0, 1],
  C[0, 1, 0],
]
```

Adjacency List 邻接表
```js
{
  'A': ['B'],
  'B': ['A', 'C'],
  'C': ['B']
}

```
